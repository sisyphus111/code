# Project Instructions

## GitHub Pages Deployment

- After any requested addition, content change, layout cleanup, or other
  modification in this repository, synchronize the completed result to the
  GitHub Pages site as part of the same task.
- The deployment is complete only after the relevant changes are committed and
  pushed to `main`, the deployed `gh-pages` branch is updated, and
  `https://sisyphus111.github.io/code/` is verified successfully.
- Skip deployment only when the user explicitly asks not to deploy for that
  particular task.
