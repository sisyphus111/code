输入输出专精

输入输出是 ACM 模式的第一道关卡，格式错误会导致 WA（Wrong Answer）。本文档深入讲解各种 IO 场景。

OJ 模式对比

LeetCode 函数模式 vs ACM 模式

特性 LeetCode 模式 ACM/OI 模式
输入处理 函数参数（自动处理） cin/scanf 手动读取
输出处理 return 返回值 cout/printf 手动输出
多组数据 函数多次调用 循环读取到 EOF
代码结构 只写核心逻辑 完整 main 函数
重要：大部分高校机考、企业笔试使用 ACM 模式，需要自己处理输入输出。

LeetCode 模式示例

class Solution {
public:
    vector twoSum(vector& nums, int target) {
        // 直接写逻辑，不管输入输出
        unordered_map map;
        for (int i = 0; i < nums.size(); ++i) {
            int complement = target - nums[i];
            if (map.count(complement)) {
                return {map[complement], i};
            }
            map[nums[i]] = i;
        }
        return {};
    }
};

ACM 模式示例（同样的问题）

#include 
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    
    int n, target;
    cin >> n >> target;  // 需要自己读取
    
    vector nums(n);
    for (int i = 0; i < n; ++i) {
        cin >> nums[i];
    }
    
    // 核心逻辑
    unordered_map map;
    for (int i = 0; i < n; ++i) {
        int complement = target - nums[i];
        if (map.count(complement)) {
            cout << map[complement] << " " << i << "\n";  // 需要自己输出
            return 0;
        }
        map[nums[i]] = i;
    }
    
    return 0;
}

读到 EOF

什么是 EOF？

EOF (End Of File) 表示输入结束。在 ACM 模式中，很多题目不会告诉你有多少组数据，需要读到 EOF 为止。

基本用法

// 读取单个整数到 EOF
int n;
while (cin >> n) {
    // 处理每个 n
    cout << n * 2 << "\n";
}

// 读取多个变量到 EOF
int a, b;
while (cin >> a >> b) {
    cout << a + b << "\n";
}

// 读取整行到 EOF
string line;
while (getline(cin, line)) {
    // 处理每行
    cout << line << "\n";
}
原理：cin >> n 在成功读取时返回 true，读到 EOF 或错误时返回 false。

实战例题：A + B Problem

题意：多组输入，每组两个整数 a 和 b，输出它们的和。

#include 
using namespace std;

int main() {
    int a, b;
    while (cin >> a >> b) {
        cout << a + b << "\n";
    }
    return 0;
}

// 输入：
// 1 2
// 3 4
// 5 6
//
// 输出：
// 3
// 7
// 11

多组输入

类型 1：给定组数

int T;
cin >> T;
while (T--) {
    // 处理每组
    int n;
    cin >> n;
    cout << n * n << "\n";
}

类型 2：读到 EOF

int n;
while (cin >> n) {
    // 处理每组
    cout << n * n << "\n";
}

类型 3：读到特定标记

// 读到 0 结束
int n;
while (cin >> n && n != 0) {
    cout << n * n << "\n";
}

// 读到 # 结束
string s;
while (cin >> s && s != "#") {
    cout << s << "\n";
}

类型 4：每组数据第一行是 n，后面 n 行数据

int n;
while (cin >> n) {
    vector a(n);
    for (int i = 0; i < n; ++i) {
        cin >> a[i];
    }
    // 处理数组 a
    int sum = accumulate(a.begin(), a.end(), 0);
    cout << sum << "\n";
}

整行读取

为什么需要 getline？

  • cin >> s 遇到空格就停止
  • getline(cin, s) 读取一整行（包含空格）
string line;
getline(cin, line);  // 读取整行，包括空格

// 示例输入："hello world"
// cin >> line 只会读到 "hello"
// getline(cin, line) 读到 "hello world"

混用 cin 和 getline 的陷阱

常见错误：cin >> n 后用 getline 会读到残留的换行符！ 
// 错误示例
int n;
cin >> n;          // 输入 "5\n"，读取 5，但 \n 残留在缓冲区
string line;
getline(cin, line); // 读到空字符串（残留的 \n）

// 正确做法 1：使用 cin.ignore()
int n;
cin >> n;
cin.ignore();      // 忽略残留的换行符
string line;
getline(cin, line);

// 正确做法 2：全部用 getline
string first_line;
getline(cin, first_line);
int n = stoi(first_line);
string second_line;
getline(cin, second_line);

按分隔符读取

// 按逗号分隔读取
string line;
getline(cin, line);
stringstream ss(line);
string token;
while (getline(ss, token, ',')) {  // 第三个参数是分隔符
    cout << token << "\n";
}

// 输入："apple,banana,cherry"
// 输出：
// apple
// banana
// cherry

输出格式陷阱

陷阱 1：行末空格

很多 OJ 会判定行末空格为格式错误（Presentation Error 或 Wrong Answer）。 
// 错误：行末有空格
for (int i = 0; i < n; ++i) {
    cout << a[i] << " ";  // 最后一个元素后也有空格
}
cout << "\n";

// 正确方法 1：判断是否是最后一个
for (int i = 0; i < n; ++i) {
    cout << a[i];
    if (i < n - 1) cout << " ";
}
cout << "\n";

// 正确方法 2：技巧性写法
for (int i = 0; i < n; ++i) {
    cout << a[i] << " \n"[i == n - 1];
}

// 正确方法 3：第一个元素前不加空格
for (int i = 0; i < n; ++i) {
    if (i > 0) cout << " ";
    cout << a[i];
}
cout << "\n";

陷阱 2：最后一行的换行符

// 有些题目要求最后一行也要换行
cout << result << "\n";  // 安全做法：总是加换行

// 有些题目要求最后一行不换行（极少数）
// 需要特判最后一组数据

陷阱 3：浮点数精度

#include 

// 错误：默认精度可能不对
double x = 3.141592653;
cout << x << "\n";  // 输出 3.14159（默认 6 位有效数字）

// 正确：指定精度
cout << fixed << setprecision(2) << x << "\n";  // 输出 3.14

// 常见要求：
// - 保留 2 位小数：fixed + setprecision(2)
// - 有效数字：setprecision(n) 不加 fixed

陷阱 4：大小写

// 题目要求输出 "YES" 或 "NO"
cout << (flag ? "YES" : "NO") << "\n";  // 正确

// 错误示例（大小写不对）
cout << (flag ? "Yes" : "No") << "\n";
cout << (flag ? "yes" : "no") << "\n";

快速 IO

为什么需要快速 IO？

当输入数据量很大（10^5 - 10^6 级别）时，cin/cout 可能超时。

方法 1：关闭同步（推荐）

#include 
using namespace std;

int main() {
    ios::sync_with_stdio(false);  // 关闭 C++ 与 C 的同步
    cin.tie(nullptr);              // 解除 cin 与 cout 的绑定
    
    // 现在 cin/cout 速度接近 scanf/printf
    int n;
    cin >> n;
    cout << n << "\n";
    
    return 0;
}
注意：使用上述优化后，不能混用 cin/cout 和 scanf/printf。

方法 2：使用 scanf/printf（C 风格）

int n;
scanf("%d", &n);
printf("%d\n", n);

// 常用格式串
scanf("%d", &n);          // int
scanf("%lld", &ll);       // long long
scanf("%lf", &d);         // double
scanf("%s", str);         // char[]

printf("%d\n", n);       // int
printf("%lld\n", ll);    // long long
printf("%.2lf\n", d);    // double，保留 2 位

速度对比

方法 相对速度 使用建议
scanf/printf 最快 简单格式，追求极致速度
cin/cout + 优化 接近 scanf 推荐，C++ 风格
cin/cout 不优化 较慢 数据量小时可用
getline 中等 需要读整行时

字符串流应用

用途 1：分割字符串

#include 

string line = "1 2 3 4 5";
stringstream ss(line);
int x;
while (ss >> x) {
    cout << x << "\n";
}

用途 2：读取不定长数组

// 一行有若干个整数，数量未知
string line;
getline(cin, line);
stringstream ss(line);
vector arr;
int x;
while (ss >> x) {
    arr.push_back(x);
}

// 现在 arr 包含所有数字

用途 3：构造字符串

stringstream ss;
ss << "x = " << x << ", y = " << y;
string result = ss.str();
cout << result << "\n";

用途 4：类型转换

// string -> int
string s = "12345";
stringstream ss(s);
int n;
ss >> n;  // n = 12345

// 或者更简单
int n = stoi(s);

// int -> string
int x = 123;
string s = to_string(x);

边界情况处理

空行处理

string line;
while (getline(cin, line)) {
    if (line.empty()) {
        // 空行
        continue;
    }
    // 处理非空行
}

前导/尾随空格

// 使用 stringstream 自动忽略前后空格
string line = "   123   ";
stringstream ss(line);
int n;
ss >> n;  // n = 123，空格被忽略

// 手动去除空格（如果需要）
string trim(string s) {
    s.erase(s.begin(), find_if(s.begin(), s.end(), [](char c) {
        return !isspace(c);
    }));
    s.erase(find_if(s.rbegin(), s.rend(), [](char c) {
        return !isspace(c);
    }).base(), s.end());
    return s;
}

Windows vs Unix 换行符

  • Unix/Linux：\n (LF)
  • Windows：\r\n (CRLF)

getline 会自动处理，但手动读字符时要小心。

读取字符

// 跳过空白字符
char c;
cin >> c;  // 跳过空格、换行

// 不跳过空白字符
char c = cin.get();  // 读取下一个字符，包括空格和换行

实战：读取不定长二维数组

// 每行数字个数不同
vector> matrix;
string line;
while (getline(cin, line)) {
    stringstream ss(line);
    vector row;
    int x;
    while (ss >> x) {
        row.push_back(x);
    }
    if (!row.empty()) {
        matrix.push_back(row);
    }
}

// 现在 matrix 包含所有数据
