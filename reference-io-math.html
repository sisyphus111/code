附录 C：输出格式控制与常用库

机考中常用的输出格式控制、数学函数和头文件速查。

1. 输出格式控制

1.1 浮点数精度控制

#include <iostream>
#include <iomanip>  // 必须包含这个头文件
using namespace std;

int main() {
    double pi = 3.14159265358979;

    // 设置小数点后位数（默认模式）
    cout << setprecision(2) << pi << '\n';  // 输出：3.1
    cout << setprecision(4) << pi << '\n';  // 输出：3.142

    // fixed：固定小数点后 n 位
    cout << fixed << setprecision(2) << pi << '\n';  // 输出：3.14
    cout << fixed << setprecision(6) << pi << '\n';  // 输出：3.141593

    // scientific：科学计数法
    cout << scientific << setprecision(3) << pi << '\n';  // 输出：3.142e+00

    return 0;
}

1.2 常用组合

用法效果示例输入输出
setprecision(2)总共 2 位有效数字3.141593.1
fixed << setprecision(2)小数点后 2 位3.141593.14
scientific << setprecision(3)科学计数法 3 位1234.51.235e+03
机考常用：cout << fixed << setprecision(2) 保留小数点后 2 位，这是最常见的要求。

1.3 整数进制与填充

#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int num = 255;

    // 进制控制
    cout << dec << num << '\n';  // 十进制：255
    cout << hex << num << '\n';  // 十六进制：ff
    cout << oct << num << '\n';  // 八进制：377

    // 显示进制前缀
    cout << showbase << hex << num << '\n';  // 0xff

    // 宽度与填充
    cout << dec;  // 切回十进制
    cout << setw(5) << 42 << '\n';         // "   42"（右对齐，宽度 5）
    cout << setw(5) << setfill('0') << 42 << '\n';  // "00042"

    // 左对齐
    cout << left << setw(5) << setfill(' ') << 42 << '\n';  // "42   "

    return 0;
}

3. cmath 数学函数库

3.1 基本数学函数

// 幂与根

// 绝对值

double b = fabs(-3.14);       // 3.14（浮点数）

// 上下取整
double x = 3.7;
double up = ceil(x);          // 4（向上取整）
double down = floor(x);       // 3（向下取整）
double nearest = round(x);    // 4（四舍五入）

// 取余
double r = fmod(5.5, 2.0);    // 1.5（浮点数取余）

5. bits/stdc++.h 万能头文件

5.1 什么是 bits/stdc++.h？

bits/stdc++.h 是 GCC 编译器提供的一个非标准头文件，包含了几乎所有 C++ 标准库。

#include <bits/stdc++.h>
using namespace std;

int main() {
    // 可以直接使用所有 STL 容器、算法、数学函数等
    vector<int> vec;
    map<string, int> mp;
    priority_queue<int> pq;
    cout << sqrt(16) << '\n';
    return 0;
}

5.2 优缺点

优点缺点
不用记忆每个库的头文件编译速度慢（包含了所有库）
竞赛和机考中方便快捷不是标准 C++，不跨平台
减少代码量工程项目中不推荐使用
机考建议：机考中可以放心使用 bits/stdc++.h，因为评测系统通常使用 GCC。但在实际项目中应该使用具体的头文件。

6. 完整示例：格式化输出

#include <iostream>
#include <iomanip>
#include <cmath>
using namespace std;

int main() {
    // 保留 2 位小数
    double pi = 3.14159265358979;
    cout << fixed << setprecision(2) << pi << '\n';  // 3.14

    // 对齐输出表格
    cout << left;
    cout << setw(10) << "Name" << setw(10) << "Score" << '\n';
    cout << setw(10) << "Alice" << setw(10) << 95 << '\n';
    cout << setw(10) << "Bob" << setw(10) << 87 << '\n';

    // 输出：
    // Name      Score
    // Alice     95
    // Bob       87

    // 填充 0
    cout << setw(5) << setfill('0') << 42 << '\n';  // 00042

    // 科学计数法
    double large = 1234567.89;
    cout << scientific << setprecision(3) << large << '\n';  // 1.235e+06

    return 0;
}
