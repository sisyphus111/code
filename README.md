## 🎓 使用建议

### 学习路径

1. **先看特辑** → 了解机考环境和输入输出
2. **顺序学习** → 按 01→18 顺序学习基础题型
3. **重点突破** → 重点攻克第 14 章动态规划
4. **实战练习** → 跟着 Hot 100 索引刷题
5. **考前复习** → 使用模板库快速回顾

### 练习方法

- **看完就写**：每个例题自己敲一遍
- **限时训练**：模拟考场环境，2-3 小时完成 3-4 题
- **总结归纳**：每做完一个题型，总结识别特征
- **反复回顾**：错题和不熟练的模板反复练习
